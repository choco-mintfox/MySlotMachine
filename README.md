# スロットマシーン

オブジェクト指向とカプセル化を意識して実装した、ブラウザで動くシンプルなスロットマシンアプリです。

## デモ（公開URL）
[アプリを開く](https://choco-mintfox.github.io/MySlotMachine/)

## 概要
画面上の「SPIN」ボタンを押すと3つのパネルが同時に回転し、それぞれの「STOP」ボタンを押すことでパネルを停止させることができます。
JavaScriptの基本的なDOM操作に加え、保守性と拡張性を意識したコード設計の練習として作成しました。

## 使用技術・環境
- **フロントエンド**: HTML5, CSS3, JavaScript (ES6+)
- **バージョン管理・ホスティング**: GitHub, GitHub Pages
